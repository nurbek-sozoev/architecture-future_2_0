Практика управления инфраструктурой через декларативные конфигурации (Terraform, Helm). Вся инфраструктура описана в коде, версионируется и деплоится автоматически.

**Бизнес-сценарии:**
- Автоматизация облачной инфраструктуры: развёртывание новых сред за минуты вместо недель
- Воспроизводимость: одинаковые среды для dev/staging/prod снижают количество инцидентов
- Масштабирование: новый домен получает инфраструктуру автоматически
