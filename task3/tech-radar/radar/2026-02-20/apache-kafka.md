Распределённая платформа потоковой обработки событий. Обеспечивает надёжную асинхронную интеграцию между доменами через pub/sub модель.

**Бизнес-сценарии:**

- Event Streaming для интеграции доменов: клиника -> ИИ (медицинские снимки), клиника -> финтех (биллинг)
- Подключение партнёров: фармкомпании и производители электроники через партнёрский домен
- Гарантия доставки сообщений и поддержка replay при сбоях
- Замена Apache Camel (O(N) вместо O(N²) соединений)
