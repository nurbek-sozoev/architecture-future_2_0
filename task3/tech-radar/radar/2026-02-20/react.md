JavaScript-фреймворк для создания современных web-приложений. Используется для замены PowerBuilder и создания портала самообслуживания.

**Бизнес-сценарии:**

- Замена PowerBuilder: современный web-клиент для операторов клиник (скорость работы +30–40%)
- Портал самообслуживания: единый UI для аналитики, дэшбордов и каталога данных
- Поддержка SSO, мобильных устройств, PWA - что невозможно с PowerBuilder
- Большой пул React-разработчиков на рынке vs. невозможность найти PowerBuilder-специалистов
