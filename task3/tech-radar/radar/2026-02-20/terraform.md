Инструмент Infrastructure as Code для декларативного управления облачной инфраструктурой. Поддерживает мультиоблачные сценарии.

**Бизнес-сценарии:**
- Автоматизация облачной инфраструктуры: Kubernetes, сети, безопасность, хранилища
- Воспроизводимость сред: одинаковая конфигурация для dev/staging/prod
- Масштабирование: новый домен или сервис получает инфраструктуру из кода за минуты
