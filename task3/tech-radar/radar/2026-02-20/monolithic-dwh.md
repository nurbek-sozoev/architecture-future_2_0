Текущая архитектура с единым хранилищем данных (DWH) на SQL Server 2008, содержащим бизнес-логику всех доменов. Рекомендуется к выводу и замене на Data Mesh + Data Lakehouse.

**Проблемы:**
- Блокирует автономность команд: любое изменение требует координации всех доменов
- Единая точка отказа: падение DWH парализует весь бизнес
- Формирование отчётов занимает часы вместо минут
- Невозможно разделить данные по доменам для compliance
