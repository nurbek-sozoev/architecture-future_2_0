Фреймворк для автоматического контроля качества данных. Определяет и проверяет expectations (ожидания) для Data Products.

**Бизнес-сценарии:**

- Контроль качества данных: финансовые отчёты для ЦБ РФ должны быть точными
- Автоматическая валидация Data Products при публикации в Lakehouse
- Compliance: SLA данных (freshness, completeness, accuracy) - обязательное требование
