Архитектурный стиль, в котором домены взаимодействуют через асинхронные события (Event Streaming). Заменяет синхронную point-to-point интеграцию на pub/sub модель.

**Бизнес-сценарии:**

- Асинхронная интеграция между доменами (клиника -> финтех, клиника -> ИИ)
- Масштабирование: новый потребитель подписывается на события без изменения источника
- Подключение партнёров (фармкомпании, производители электроники) через событийную модель
