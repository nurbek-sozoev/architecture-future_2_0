Централизованный API Gateway для управления внешними интеграциями. Обеспечивает аутентификацию, rate limiting, мониторинг и стандартизированный интерфейс для партнёров.

**Бизнес-сценарии:**
- Интеграция фармкомпаний: стандартизированный API для получения справочников лекарств
- Интеграция производителей электроники: каталоги оборудования, сервисные контракты
- Подключение нового партнёра за дни вместо месяцев через стандартизированный интерфейс
- Централизованная безопасность для всех внешних интеграций
