Платформа оркестрации data pipelines. Управляет расписанием, зависимостями и мониторингом ETL/ELT-процессов в Data Lakehouse.

**Бизнес-сценарии:**

- Оркестрация data pipelines для Data Lakehouse: загрузка Data Products, трансформации, агрегации
- Мониторинг и алертинг при сбоях пайплайнов - критично для своевременности отчётности
- Управление зависимостями между пайплайнами разных доменов
